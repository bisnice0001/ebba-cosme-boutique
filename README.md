# Ebba Cosme by Hayyin

A premium, responsive landing page for **Ebba Cosme by Hayyin**—a boutique salon and spa located in Karangploso, Malang, Indonesia. This site is designed with a "soft feminine luxury" aesthetic, optimized for high conversion, search engine discoverability (SEO), and fast loading times.

---

## 🌟 Features

- **Soft Feminine Luxury Aesthetic**: Curated color palette (mauves, soft roses, dark plum) matching the salon's physical interior.
- **Dynamic Pricing & Treatment Catalog**: Filterable list of treatments (Hair, Skin, Nails, Body, Spa) with clean, transparent pricing.
- **WhatsApp Integration**: Floating button and booking CTAs that directly pre-fill a template message to the salon's admin.
- **Performance Optimized**: Built using Next.js with lazy loading (`next/dynamic`), modern image formats (`.webp`), and minimal blocking scripts.
- **SEO & Accessibility**: Complete metadata, structured schema, proper heading structure, and color contrast ratios exceeding WCAG AA standards.

---

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS & Vanilla CSS
- **Icons**: Lucide React

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js and npm/bun installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/danusyarifudinnn/ebbacosme.git
   cd ebba-cosme-boutique
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser to view the site.

### Build and Deploy

To create a production build:
```bash
npm run build
npm start
```
